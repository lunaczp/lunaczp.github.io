◆ row_drop_table_for_mysql_in_background()

static

Drops a table for MySQL as a background operation. MySQL relies on Unix in ALTER TABLE to the fact that the table handler does not remove the table before all handles to it has been removed. Furhermore, the MySQL's call to drop table must be non-blocking. Therefore we do the drop table as a background operation, which is taken care of by the master thread in srv0srv.cc.

Returns
error code or DB_SUCCESS
Parameters
namein: table name
