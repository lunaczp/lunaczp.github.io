◆ mtr_memo_note_modifications()

static void mtr_memo_note_modifications ( mtr_t *  mtr)
static

Add the modified pages to the buffer flush list. They are released in the order opposite to which they were pushed to the memo. NOTE! It is essential that the x-rw-lock on a modified buffer page is not released before buf_page_note_modification is called for that page! Otherwise, some thread might race to modify it, and the flush list sort order on lsn would be destroyed.

mtrin: mtr
