934 /*
935  variance(a) =
936 
937  = sum (ai - avg(a))^2 / count(a) )
938  = sum (ai^2 - 2*ai*avg(a) + avg(a)^2) / count(a)
939  = (sum(ai^2) - sum(2*ai*avg(a)) + sum(avg(a)^2))/count(a) =
940  = (sum(ai^2) - 2*avg(a)*sum(a) + count(a)*avg(a)^2)/count(a) =
941  = (sum(ai^2) - 2*sum(a)*sum(a)/count(a) + count(a)*sum(a)^2/count(a)^2 )/count(a) =
942  = (sum(ai^2) - 2*sum(a)^2/count(a) + sum(a)^2/count(a) )/count(a) =
943  = (sum(ai^2) - sum(a)^2/count(a))/count(a)
944 
945 But, this falls prey to catastrophic cancellation. Instead, use the recurrence formulas
946 
947  M_{1} = x_{1}, ~ M_{k} = M_{k-1} + (x_{k} - M_{k-1}) / k newline
948  S_{1} = 0, ~ S_{k} = S_{k-1} + (x_{k} - M_{k-1}) times (x_{k} - M_{k}) newline
949  for 2 <= k <= n newline
950  ital variance = S_{n} / (n-1)
951 
952 */
