Detailed Description

template<typename Type, int N = IB_N_SLOTS, template< typename, int > class Indexer = thread_id_indexer_t>
class ib_counter_t< Type, N, Indexer >

Class for using fuzzy counters. The counter is not protected by any mutex and the results are not guaranteed to be 100% accurate but close enough. Creates an array of counters and separates each element by the CACHE_LINE_SIZE bytes
