62 /*
63  Temporary table used by semi-join DuplicateElimination strategy
64 
65  This consists of the temptable itself and data needed to put records
66  into it. The table's DDL is as follows:
67 
68  CREATE TABLE tmptable (col VARCHAR(n) BINARY, PRIMARY KEY(col));
69 
70  where the primary key can be replaced with unique constraint if n exceeds
71  the limit (as it is always done for query execution-time temptables).
72 
73  The record value is a concatenation of rowids of tables from the join we're
74  executing. If a join table is on the inner side of the outer join, we
75  assume that its rowid can be NULL and provide means to store this rowid in
76  the tuple.
77 */
