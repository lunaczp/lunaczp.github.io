◆ m_free_limit

ulint AbstractCallback::m_free_limit
protected

Minimum page number for which the free list has not been initialized: the pages >= this limit are, by definition, free; note that in a single-table tablespace where size < 64 pages, this number is 64, i.e., we have initialized the space about the first extent, but have not physically allocted those pages to the file.

See also
FSP_LIMIT.
