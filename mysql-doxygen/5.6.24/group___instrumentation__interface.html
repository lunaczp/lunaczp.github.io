◆ PSI_FLAG_GLOBAL

#define PSI_FLAG_GLOBAL   (1 << 0)

Global flag. This flag indicate that an instrumentation point is a global variable, or a singleton.

◆ PSI_FLAG_MUTABLE

#define PSI_FLAG_MUTABLE   (1 << 1)

Global flag. This flag indicate that an instrumentation point is a general placeholder, that can mutate into a more specific instrumentation point.
