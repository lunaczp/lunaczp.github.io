◆ m_rwlock_array

my_atomic_rwlock_t PFS_atomic::m_rwlock_array
staticprivate

Internal rwlock array. Using a single rwlock for all atomic operations would be a bottleneck. Using a rwlock per performance schema structure would be too costly in memory, and use too many rwlock. The PFS_atomic implementation computes a hash value from the atomic variable, to spread the bottleneck across 256 buckets, while still providing –transparently for the caller– an atomic operation.
