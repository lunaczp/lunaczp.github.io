122 /*
123  * el_set/el_get parameters
124  *
125  * When using el_wset/el_wget (as opposed to el_set/el_get):
126  * Char is wchar_t, otherwise it is char.
127  * prompt_func is el_wpfunc_t, otherwise it is el_pfunc_t .
128 
129  * Prompt function prototypes are:
130  * typedef char *(*el_pfunct_t) (EditLine *);

133  * For operations that support set or set/get, the argument types listed are for
134  * the "set" operation. For "get", each listed type must be a pointer.

137  * Operations that only support "get" have the correct argument types listed.
